# 👵👴 어르신을 위한 출석부 사용 설명서

안녕하세요! '몸펴기생활운동 연신내 출석부' 사용법을 쉽고 간단하게 알려드릴게요.

---

## 1. 앱 설치하기 (바탕화면에 바로가기 만들기)

매번 인터넷 주소를 치고 들어오기 번거로우시죠? 아래 방법으로 스마트폰 바탕화면에 아이콘을 만들어두면, 다음부터는 아이콘만 눌러서 바로 들어올 수 있어요.

1.  **스마트폰 인터넷(크롬 또는 사파리)으로 출석부 사이트에 접속**합니다.
2.  화면 오른쪽 위 또는 아래에 있는 **점 세 개(⋮) 또는 공유(↑) 버튼**을 누릅니다.
3.  메뉴에서 **'홈 화면에 추가'** 또는 **'앱 설치'**를 찾아 누릅니다.
4.  바탕화면에 출석부 아이콘이 생기면 성공!

---

## 2. 출석 체크하기 (회원용)

수련하러 오셨을 때, 아래 두 가지 방법 중 편한 방법으로 출석 체크를 하시면 됩니다.

### 방법 1: 직접 이름 검색하기

1.  앱 첫 화면에서 **[회원 출석]** 버튼을 누릅니다.
2.  **오늘 날짜와 현재 수업이 맞는지 확인**합니다. (보통 자동으로 맞춰져 있어요)
3.  **'이름 또는 초성 검색'** 칸에 본인 이름(예: 김철수)이나 이름의 첫 글자(예: ㄱㅊㅅ)를 씁니다.
4.  아래에 나타난 **본인 이름 버튼을 누르면 출석 끝!** "✅ 출석 완료!" 메시지가 나오면 성공입니다.

### 방법 2: QR 코드로 출석하기

도장에 비치된 수업별 QR 코드를 스마트폰 카메라로 찍어서 출석하는 방법입니다.

1.  **스마트폰 카메라**를 켜고, 현재 참여하는 수업의 **QR 코드를 비춥니다.**
2.  화면에 나타나는 **인터넷 주소 링크를 누릅니다.**
3.  출석부 앱이 열리면서 수업이 자동으로 선택됩니다.
4.  **'이름 또는 초성 검색'** 칸에 본인 이름을 검색하고, **이름 버튼을 누르면 출석 끝!**

---

## 3. 관리자 기능 사용하기 (관리자용)

회원을 관리하거나 통계를 보려면 관리자 기능이 필요해요.

### 가. 관리자 로그인

1.  앱 첫 화면에서 **[관리자 전용]** 버튼을 누릅니다.
2.  비밀번호(초기 비밀번호: `1234`)를 입력하고 **[로그인]** 버튼을 누릅니다.

### 나. 회원 관리 (등록 및 삭제)

*   **새 회원 등록:** 하단 메뉴에서 **[회원관리]**를 누르고, 상단에서 이름과 전화번호를 입력 후 **[등록하기]** 버튼을 누릅니다.
*   **회원 삭제:** 목록에서 삭제할 회원 옆의 **[삭제]** 버튼을 누릅니다. (⚠️ 주의: 출석 기록도 함께 삭제됩니다!)

### 다. 일괄 출석 입력 (명부 입력)

종이 명부를 보고 한번에 출석을 입력하거나, 스캔한 명단을 붙여넣을 때 사용합니다.

1.  하단 메뉴에서 **[일괄출석]**을 누릅니다.
2.  **날짜와 수업을 선택**합니다.
3.  **'명단 입력'** 칸에 출석한 회원들의 이름을 한 줄에 한 명씩 입력합니다.
4.  **[일괄 출석 처리]** 버튼을 누르면 출석이 완료됩니다.

### 라. QR 코드 관리 (인쇄하기)

회원들이 QR 코드로 출석할 수 있도록, 수업별 QR 코드를 인쇄해서 비치해두세요.

1.  하단 메뉴에서 **[설정]**을 누릅니다.
2.  **[차수별 QR 코드 출력]** 버튼을 누릅니다.
3.  **[인쇄하기]** 버튼을 눌러 모든 수업의 QR 코드를 인쇄합니다.

### 마. 리포트(통계) 보기

하단 메뉴에서 **[리포트]**를 누르면 월별 출석 현황, 출석 순위 등 다양한 통계를 볼 수 있어요.

### 바. 데이터 관리 (백업 및 복구)

혹시 모를 상황에 대비해 데이터를 안전하게 보관하는 기능이에요.

1.  하단 메뉴에서 **[설정]**을 누릅니다.
2.  **[데이터 관리 (백업/복구)]** 버튼을 누릅니다.
3.  **[백업 파일 생성]**을 눌러 데이터를 컴퓨터나 스마트폰에 저장해두세요. (비밀번호를 꼭 기억하세요!)
4.  문제가 생겼을 때 **[백업 파일에서 복구]**를 눌러 이전에 저장해둔 파일로 되돌릴 수 있습니다.

### 사. 데이터 보정하기 (문제가 있을 때만 사용)

회원 목록에 'Invalid Date 가입'이라고 보이거나, 리포트가 이상하게 보일 때 사용하세요.

1.  **[설정]** > **[데이터 관리]**로 들어갑니다.
2.  **[가입일 데이터 복구]**와 **[출석 기록 데이터 복구]** 버튼을 차례대로 눌러주세요. 잘못된 데이터를 시스템이 알아서 고쳐줍니다.
